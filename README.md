# 🛍️ Product Filter UI

A responsive product filter interface built with **React** and **TailwindCSS**.  
It features:

- 🔍 Live search by product name  
- 🏷️ Category filters (brand, color, price)  
- 🧱 Reusable card components  
- 📱 Fully responsive layout
